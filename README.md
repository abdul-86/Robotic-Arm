\# ESP32 Based Robotic Arm



\## 📌 Project Description

This project is an ESP32 based robotic arm controlled using servo motors. 

The robotic arm is designed using 3D printed parts and a metallic base plate for stability.



\## ⚙️ Hardware Used

\- ESP32

\- MG996R Servo Motors

\- 3D Printed Robotic Arm

\- Metallic Base Plate

\- External Power Supply



\## 💻 Software Used

\- Arduino IDE

\- Embedded C



\## 🚀 System Implementation

The system is 50% implemented. 

Servo motors are connected and tested. 

Base structure is completed.
